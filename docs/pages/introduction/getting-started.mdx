# 开始使用

欢迎来到 **Aleo 中文文档**。本节会帮助你理解 Aleo 的基本概念，并选择适合自己的入门路径。

Aleo 是一个以隐私为核心的区块链平台。开发者可以使用 Leo 编程语言编写应用，在保护用户数据的同时，让程序逻辑和状态转换能够被验证。

## 选择你的入门路径

如果你是第一次接触 Aleo，建议按下面的顺序阅读：

  <Card title="快速开始" href="/introduction/quick-start" />
  <Card title="安装" href="/introduction/installation" />
</Cards>

## 你将学到什么

通过本节内容，你会了解：

- Aleo 的开发入口和基础工具。
- 如何使用 Leo Playground 快速体验 Aleo 程序。
- 如何在本地安装 Leo 和 snarkOS。
- 如何继续学习 Leo 语言、CLI、devnet 和部署流程。

## 推荐顺序

### 1. 快速开始

如果你想先跑通完整流程，请从 [快速开始](/introduction/quick-start) 开始。该指南会带你使用浏览器中的 Leo Playground 创建程序、部署程序并执行函数。

这种方式不需要先配置完整本地环境，适合第一次体验 Aleo 的开发流程。

### 2. 安装本地环境

如果你准备在本地开发，请继续阅读 [安装](/introduction/installation)。该页面会介绍如何安装 Git、Rust、Leo 和 snarkOS，并验证命令行工具是否可用。

本地环境更适合长期开发、测试和调试，也方便你使用 devnet 在部署到测试网之前验证程序。

## 后续学习

完成入门步骤后，可以继续阅读：

- [Leo CLI guide](https://docs.leo-lang.org/cli/overview)：学习 Leo 命令行工具。
- [Leo Language guide](https://docs.leo-lang.org/language/overview)：学习 Leo 语言语法和核心概念。
- [Local devnet guide](https://docs.leo-lang.org/guides/devnet)：在本地网络中测试程序。
- [Aleo Developer Docs](https://developer.aleo.org/)：查看 Aleo 官方开发文档。

如果你正在参加 Aleo 101 Bootcamp，可以先完成课程中的章节任务，再回到文档中补充理解相关概念。
