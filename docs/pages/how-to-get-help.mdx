# 如何获取帮助

Discord 是 Aleo 与社区沟通的主要渠道。如果你的问题没有被当前开发者文档覆盖，可以加入 Discord，并前往工程支持相关频道寻求帮助。

你也可以参加社区会议。这些会议面向所有人开放，并在 Discord 上举办。具体日期和时间请查看 Discord 服务器中的 Events 区域。

目前文档中列出的会议包括：

- Developers Calls：每月第一和第三个星期一举行，时间为太平洋时间 7:30 AM 到 8:30 AM，主要供开发者讨论技术主题和更新。
- Community Call：每月最后一个星期一举行，时间为太平洋时间 8:00 AM 到 9:00 AM，主要面向整个社区同步 Aleo 生态的最新进展。

## 建议的求助方式

- 先搜索文档，确认问题是否已经有现成答案。
- 如果是开发问题，尽量带上清晰的上下文，例如命令、报错信息、复现步骤和你已经尝试过的方法。
- 如果问题与某个仓库或 PR 有关，附上链接会更容易得到有效帮助。

## 支持入口

- Discord：Aleo 社区和工程支持的主入口。
- Community Calls：适合了解整体动态和公开讨论。
- 开发者文档：适合先进行自助排查。

如果你正在参加 Bootcamp，通常也可以先查看课程资料、作业仓库和群内已整理信息，再决定是否升级到 Discord 提问。
