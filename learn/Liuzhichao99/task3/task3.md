# Task 3 - Private Multiplier Demo

## 项目概述

本项目实现了基于 Leo 编程语言的私密乘法小应用，展示了 Aleo 隐私计算的核心特性。

## 核心功能

### 1. 私密乘法运算
- **函数**: `private_multiply(a, b)`
- **输入**: 两个私密字段值 `a` 和 `b`
- **输出**: 乘法结果 `c = a * b`
- **特点**: 输入值保持私密，只输出计算结果

### 2. 简洁的用户界面
- **技术栈**: HTML5 + CSS3 + JavaScript
- **设计风格**: 现代化渐变背景，毛玻璃效果
- **交互方式**: 输入框 + 按钮操作
- **响应式设计**: 适配不同屏幕尺寸

### 3. 本地模拟执行
- **目标**: 便于理解和演示隐私计算原理
- **扩展性**: 可轻松接入真实 Aleo 区块链
- **调试友好**: 支持快速迭代和测试

## 技术亮点

## 实现细节

### Leo 合约设计
```leo
function private_multiply:
    input a as private field,
    input b as private field
    output c as field
{
    c = a * b;
    return c;
}
```

### 前端交互流程
1. 用户输入两个数字
2. 点击执行按钮触发运算
3. 显示加载动画和进度提示
4. 返回私密乘法结果
5. 展示计算过程的私密性说明
