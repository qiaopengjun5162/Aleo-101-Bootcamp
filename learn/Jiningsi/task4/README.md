\# 🔞 隐私年龄验证 dApp



基于 Aleo 零知识证明技术的隐私年龄验证应用



\## 项目介绍



这是一个典型的零知识证明应用场景：\*\*验证用户是否年满18岁，但不泄露用户的真实年龄\*\*。



利用 Aleo 区块链的 ZK-SNARKs 技术，年龄数据全程加密，验证过程只返回「是/否」的结果，永远不会暴露具体年龄数值。



\## 核心特性



\- 🔒 \*\*零知识证明\*\*：使用 ZK-SNARKs 技术保护年龄隐私

\- 🕵️ \*\*隐私保护\*\*：验证者只知道结果，无法获知真实年龄

\- ✅ \*\*可验证\*\*：验证结果公开透明，不可篡改

\- ⛓️ \*\*去中心化\*\*：运行在 Aleo 区块链上



\## 功能说明



1\. \*\*用户注册\*\*：输入ID和年龄，数据加密上链

2\. \*\*ZK 验证\*\*：执行零知识证明验证

3\. \*\*结果查询\*\*：只返回「成年/未成年」，不泄露年龄

4\. \*\*交易日志\*\*：完整记录每笔 ZK 证明交易



\## 运行方式



直接在浏览器打开 `index.html` 即可体验完整功能。



\## 技术栈



\- Leo 编程语言（Aleo 智能合约）

\- HTML/CSS/JavaScript（前端）

\- 零知识证明（ZK-SNARKs）
