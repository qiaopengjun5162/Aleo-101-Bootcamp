# 入门指南（Getting Started）

请选择适合您应用场景的工具链：

 - Leo — 使用 Leo 编写、编译并部署 Aleo 程序：这是一种用于零知识证明应用的高级编程语言。

 - SDK — 使用 Provable SDK 构建能够与 Aleo 网络交互的 JavaScript/TypeScript 应用程序。

-  钱包适配器 (Wallet Adapter) — 将 Aleo 钱包连接功能集成到您的 React 前端中。

- 嵌入式钱包 (Embedded Wallets) — 在您的前端中集成由 Dynamic 支持的嵌入式 Aleo 钱包。
