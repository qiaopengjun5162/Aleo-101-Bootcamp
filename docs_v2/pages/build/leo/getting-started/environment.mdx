# 您的开发环境
开发者可以选择多种多样的开发环境。

## 插件
Leo 团队为多款编辑器提供了插件支持。如果您没有在列表中看到您喜欢的编辑器，请通过 GitHub（https://github.com/ProvableHQ/leo/issues/new） 与我们联系。

### VS Code

点击此处下载编辑器：https://code.visualstudio.com/download

*安装*： 

1.在 VS Code 市场（Marketplace）中搜索并安装 Leo。

2.正确的扩展 ID 为`aleohq.leo-extension`，描述应注明为 "the official VSCode extension for Leo"（Leo 官方 VSCode 扩展）。

*使用方法*：

1.打开 VS Code。

2.进入“设置” > “扩展”，或使用左侧面板的“扩展”按钮来启用 Leo 插件。

### Sublime Text
点击此处下载编辑器：https://www.sublimetext.com/download。 Sublime 的 LSP（语言服务器协议）插件对 Aleo 指令的支持是通过一个语言服务器（language-server）来实现的。

*安装*：

1.通过 Package Control 安装 LSP 和 LSP-leo。

2.重启 Sublime。

*使用方法*

按照以下步骤切换 Leo 的语法高亮、悬停提示和标记（tokens）功能：

1.打开`Sublime Text`。

2.依次点击：设置 (Settings) > 选择配色方案... (Select Color Scheme...) > LSP-leo


### IntelliJ
点击此处下载编辑器：https://www.jetbrains.com/idea/download/

安装： 在您的 IDE 中安装并启用 Leo 插件(https://plugins.jetbrains.com/plugin/19979-leo)。
