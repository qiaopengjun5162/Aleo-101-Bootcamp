### Shield Wallet 使用教程（Aleo 新手版）

如果你准备开始参与 Aleo 生态，或者正在参加 Aleo-101 Bootcamp，那么第一步通常就是安装并使用钱包。
目前 Aleo 生态中最常用的钱包之一，就是 **Shield Wallet**。 

1. #### 什么是 Shield Wallet？

  **Shield Wallet** 是 **Aleo 生态**中的主流钱包之一。
  它支持：

  * 创建 Aleo 地址
  * 管理资产
  * 签名交易
  * 连接 Aleo dApp
  * Testnet 测试
  * 开发者交互

2. #### 安装 Shield Wallet

  建议通过官方渠道安装浏览器插件，shield钱包官网地址: https://aleo.org/shield/。
<div><img src="public/shield.jpg" /></div>

安装完成后：

1. 打开插件
2. 点击Create Wallet
3. 设置钱包密码
4. **保存助记词（Seed Phrase）**
5. 完成验证



3. #### 创建钱包注意事项

  创建钱包时，系统会生成一组助记词，这非常重要。
  **请注意：**

  * ⚠️ 助记词 = 钱包所有权

  * ⚠️ 丢失无法找回

  * ⚠️ 不要截图保存

  * ⚠️ 不要发送给任何人

    

    **建议**：

  * 抄写到纸上
  * 离线保存
  * 不要存聊天软件

  

4. #### 切换 Aleo Testnet

  参与 Bootcamp 或测试应用时，一般需要使用 Testnet。
  在 Shield 中：

  1. 打开钱包
  2. 进入 设置 -> 偏好设置 -> 网络
  3. 选择 Aleo Testnet
  4. 切换完成后，你就可以开始测试链上的操作。

  

5. #### 获取测试币（Faucet）

  Testnet 需要测试币才能发送交易, 领取测试币地址: https://faucet.aleo.org/
<div><img src="public/faucet.jpg" /></div>

获取方式通常是：

1. 打开 Faucet 页面
2. 链接钱包
3. 领取即可
4. 领取成功后也可以在区块链浏览器上查看这笔交易：https://testnet.explorer.provable.com/
